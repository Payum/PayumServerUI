PayumServerUI
=============

The client for [PayumServer](https://github.com/Payum/PayumServer).

## Use container

```bash
$ docker run --rm -p 80:80 payum/server-ui
```

## Use Locally

```bash
bower install
http-server -p 8100 -c-1
```

Open in browser `localhost:8100` and enter the url of the PayumServer.
